Simulaciones Numéricas de Física

Repositorio original en VB6 y migración completa a HTML5 + JavaScript estático, publicado desde el mismo repositorio con GitHub Pages.

Este sitio conserva el carácter de laboratorio educativo del proyecto original: ecuaciones simples, trazas cualitativas y controles directos. Las simulaciones corren enteramente del lado del cliente.

Publicación

El sitio se despliega por GitHub Actions en GitHub Pages. No necesita build, backend ni dependencias externas para funcionar.

Todo lo que ves acá es estático: HTML, CSS y JavaScript.

Referencia

La página de referencia documenta el mapeo VB6 → web, los parámetros de cada simulación y el criterio usado para mantener el comportamiento cualitativo del original.
