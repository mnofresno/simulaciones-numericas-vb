# Simulaciones Numericas

Este repositorio conserva los proyectos originales en Visual Basic 6.0 y agrega una migracion estatica a HTML5 + JavaScript publicada con GitHub Pages.

## GitHub Pages

El sitio publico queda organizado en tres paginas:

- `index.html`: portada y acceso rapido.
- `simulaciones.html`: SPA con las 7 simulaciones migradas.
- `referencia.html`: documentacion del mapeo VB6 -> web.

## Simulaciones incluidas

1. Ondas Longitudinales
2. Ondas Transversales
3. Figuras de Lissajous
4. Pendulo Elastico
5. Orbitas Planetarias
6. Fluido Viscoso
7. Ley de Snell

## Notas

- Todo el sitio es estatico. No requiere backend ni build.
- Los proyectos VB6 originales siguen presentes en el repo como referencia historica.
- Para ejecutar el software VB6 original todavia hacen falta los runtimes de Visual Basic 6.0:
  https://www.microsoft.com/en-us/download/details.aspx?id=24417
